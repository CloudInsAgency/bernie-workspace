## Key Decisions

- **Light theme only** — dark mode and ThemeToggle component removed entirely. Decision based on competitor analysis (all major deal sites use light theme; dark signals "tech tool" not "shopping site").

- **CSS variables over hardcoded colors** — every color uses `var(--bg-main)`, `var(--accent)`, etc. Makes future palette changes a one-file edit.

- **Domain canonical: non-www** — `cloudpricedeals.com` is primary, `www` 308-redirects to it. Matches sitemap canonical URLs.

- **Google Search Console = Domain property** (not URL prefix) — verified via Namecheap DNS TXT after HTML tag and HTML file methods both failed.

- **Sitemap as Next.js sitemap.js** (not static XML) — auto-generated at `/sitemap.xml` on every deploy from `src/app/sitemap.js`.

- **GitHub web editor workflow** preferred for quick edits; Claude Code on local Mac for larger refactors. Avoids force-push incidents from past terminal git work.

- **Comparison pages are dynamic routes** (`/compare/[slug]`) backed by `src/data/comparisons.js` array — same pattern as `/guides/[id]` for scalability.

- **Facebook Page is separate** for CloudPriceDeals — not run from Cloud Insurance Agency page (Craig's other business).

- **No Meta Pixel installed yet** — prepared but waiting on actual Pixel ID before Claude Code installs it.
